Open Source

I build and contribute to open-source tools and platforms that power neuroscience research worldwide. From web-based data portals to signal processing toolkits, my work focuses on making EEG research infrastructure scalable, accessible, and FAIR-compliant.

Core Developer & Infrastructure Lead

NEMAR

nemar.org

The Neuroelectromagnetic Data Archive and Repository — an open-access platform for archiving, sharing, and processing neuroelectromagnetic data at scale. NEMAR provides a web interface for data exploration, automated BIDS validation, cloud-based EEG processing pipelines via the Neuroscience Gateway (NSG), and integration with OpenNeuro for long-term archival. I designed and built the full-stack web application, data ingestion pipelines, and HPC compute integration that serve the neuroscience community.

EEGDash

eegdash.org

An interactive web dashboard for real-time EEG data exploration, quality assessment, and visualization. EEGDash provides researchers with an intuitive browser-based interface to inspect EEG recordings, view channel topographies, and assess data quality metrics — no MATLAB or Python installation required.

Braindecode

braindecode.org

An open-source Python library for EEG deep learning built on PyTorch and compatible with MNE-Python. Braindecode provides state-of-the-art deep learning architectures for EEG decoding (EEGNet, ShallowNet, Deep4Net, etc.), along with data loading utilities for common BCI and neuroscience datasets. I contribute to expanding its model zoo, self-supervised learning features, and dataset integrations.

Python PyTorch MNE-Python Deep Learning
Docs GitHub
Developer & Maintainer

EEGLAB

eeglab.org

One of the most widely used open-source toolboxes for EEG signal processing, used by over 100,000 researchers worldwide. EEGLAB provides functions for data import, artifact rejection, ICA decomposition, time-frequency analysis, and statistical testing. I contribute to the EEGLAB ecosystem including the HPC portal interface, BIDS-EEG integration, HED annotation tools, and the Python bridge (ICLabel in Python).

MATLAB Python Signal Processing ICA
Website GitHub
Co-Organizer

NeurIPS 2025 EEG Foundation Challenge

eeg2025.github.io

Co-organized the EEG Foundation Challenge accepted to the NeurIPS 2025 Competition Track. The challenge addressed cross-task transfer learning and subject-invariant representation in EEG decoding, using the large-scale HBN-EEG dataset (3,000+ participants, 6 cognitive tasks). Built the challenge infrastructure including data distribution, evaluation pipelines, and leaderboard.

Infrastructure Philosophy

I believe that great neuroscience research requires great infrastructure. My open-source work focuses on closing the gap between raw EEG data and scientific insight — building web platforms for data management (NEMAR), interactive exploration tools (EEGDash), standardized processing pipelines (EEGLAB + BIDS), and deep learning toolkits (Braindecode) that make it possible for any researcher to work with large-scale EEG datasets without reinventing the wheel.

I work across the full stack: from HPC job scheduling and cloud compute orchestration to frontend visualization and REST API design. I am particularly passionate about FAIR data principles and building systems that make neuroscience data findable, accessible, interoperable, and reusable.
